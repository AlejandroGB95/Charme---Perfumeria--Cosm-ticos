

# 🌟 Charme | Perfumería de Lujo en Ronda

Bienvenido al repositorio oficial de la página web de **Charme**, una perfumería y boutique de cosmética ubicada en Ronda, Málaga. Este proyecto ha sido diseñado para ofrecer una experiencia de usuario elegante, minimalista y funcional, adaptada a las necesidades de un negocio local premium.

---

## 🎨 Sobre el Proyecto
Charme es una identidad visual que combina la elegancia clásica con la modernidad digital. La web está construida con un enfoque en la rapidez de carga y la facilidad de contacto, permitiendo a los clientes locales descubrir productos exclusivos y recibir asesoría directa.

### Características Principales:
- **Diseño "Premium Minimalist":** Uso de tipografías serif (`Playfair Display`) y paleta de colores en tonos lavanda y carbón.
- **Botón de WhatsApp Flotante:** Conexión directa y rápida para asesoría personalizada.
- **Sección de Ubicación Dinámica:** Integración con Google Maps para guiar a los clientes a la Calle Granada.
- **Catálogo de Marcas en Texto:** Una sección elegante que resalta las firmas exclusivas (Chanel, Dior, Loewe, etc.) sin sobrecargar la web de imágenes.
- **Formulario de Contacto:** Integración con Formspree para recibir solicitudes de asesoría directamente en el email de la tienda.
- **Totalmente Responsiva:** Optimizada para dispositivos móviles (perfecta para verla mientras caminas por Ronda).

---

## 🛠️ Tecnologías Utilizadas
Para este proyecto se han utilizado tecnologías web estándar para asegurar compatibilidad total y facilidad de mantenimiento:
- **HTML5:** Estructura semántica para mejor SEO local.
- **CSS3:** Estilos personalizados, incluyendo Flexbox y animaciones de transición.
- **FontAwesome:** Iconografía profesional para redes sociales y contacto.
- **Google Fonts:** Tipografía de lujo integrada.
